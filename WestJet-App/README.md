# WestJet App

## Overview
The WestJet App is a TypeScript-based application designed to provide a seamless experience for users. This project serves as a demonstration of modern web application development practices, utilizing TypeScript for type safety and clarity.

## Project Structure
```
WestJet-App
├── src
│   ├── app.ts          # Entry point of the application
│   └── types
│       └── index.ts    # Type definitions used throughout the application
├── package.json         # npm configuration file
├── tsconfig.json        # TypeScript configuration file
└── README.md            # Project documentation
```

## Installation

To get started with the WestJet App, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Anant2510/WestJet-App.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd WestJet-App
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

## Usage

To run the application, use the following command:

```bash
npm start
```

This will start the application and you can access it at `http://localhost:3000` (or the specified port in your configuration).
